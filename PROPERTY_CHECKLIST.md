# How to Add a Property - Quick Checklist

Follow this checklist when adding a new property to ensure nothing is missed.

## Before You Start

- [ ] Have the property details ready
- [ ] Prepare the 360° virtual tour embed URL
- [ ] Take note of all amenities
- [ ] Confirm pricing and availability

## Step 1: Access the CMS

- [ ] Log in to the admin panel at `/admin/login`
- [ ] Navigate to Properties → Add New Property

## Step 2: Basic Information

- [ ] Enter a descriptive **Title**
  - Include: BHK/size, property type, location
  - Example: "3BHK Premium Apartment in OMR"
- [ ] Verify the auto-generated **Slug** (or customize it)
- [ ] Select **Deal Type**: Rent or Lease
- [ ] Select **Usage Type**: Residential or Commercial
- [ ] Enter **Property Subtype** (e.g., Apartment, Villa, Office)
- [ ] Enter **Area** (e.g., OMR, ECR, Velachery)

## Step 3: Pricing & Specifications

- [ ] Enter **Price** (monthly amount in INR)
- [ ] Enter **Size** (in square feet)
- [ ] Enter **Bedrooms** (if applicable)
- [ ] Enter **Bathrooms**
- [ ] Enter **Parking** details (e.g., "2 covered")

## Step 4: 360° Virtual Tour

- [ ] Paste the **Embed URL** for the 360° tour
  - YouTube: `https://www.youtube.com/embed/VIDEO_ID`
  - Matterport: Use their embed URL
  - Other platforms: Use iframe embed URL
- [ ] Verify the **Preview** loads correctly
- [ ] Ensure the tour is fullscreen-capable

## Step 5: Amenities

- [ ] Select all applicable amenities by clicking them
- [ ] Common amenities to consider:
  - [ ] Lift
  - [ ] Parking
  - [ ] Security
  - [ ] Power Backup
  - [ ] Water Supply
  - [ ] Gym
  - [ ] Swimming Pool
  - [ ] Garden
  - [ ] Play Area
  - [ ] Club House
  - [ ] CCTV
  - [ ] Intercom

## Step 6: Publishing

- [ ] Check **"Publish this property"** to make it live
- [ ] Optionally check **"Feature on homepage"** for premium properties
  - Only feature your best 2-3 properties at a time

## Step 7: Review & Save

- [ ] Double-check all information for accuracy
- [ ] Verify pricing is correct
- [ ] Confirm the 360° tour URL works
- [ ] Click **"Create Property"**

## Step 8: Verify on Public Site

- [ ] After creation, click **"Preview"** or visit the public site
- [ ] Verify the property appears in listings
- [ ] Test the 360° tour loads properly
- [ ] Check that all details display correctly
- [ ] Test WhatsApp and Call buttons

## Optional: Share the Property

- [ ] Click **"Copy Link"** to get the property URL
- [ ] Share on social media (Instagram, Facebook, etc.)
- [ ] Add to your Instagram bio link (if featured)
- [ ] Send to potential clients via WhatsApp

## Common Mistakes to Avoid

❌ **Don't**:
- Use vague titles like "Nice Property" or "For Rent"
- Forget to add the 360° tour embed URL
- Leave amenities unselected
- Publish without previewing first
- Use the YouTube watch URL instead of embed URL

✅ **Do**:
- Be specific and descriptive in titles
- Test the embed URL before saving
- Select all relevant amenities
- Preview before publishing
- Keep information accurate and up-to-date

## Quick Reference: Embed URL Formats

### YouTube
```
https://www.youtube.com/embed/VIDEO_ID
```

### Matterport
```
https://my.matterport.com/show/?m=SHOWCASE_ID
```

### Kuula
```
https://kuula.co/share/TOUR_ID?fs=1&vr=1&sd=1&thumbs=1
```

## Need to Update Later?

1. Go to Properties list
2. Click "Edit" next to the property
3. Make your changes
4. Click "Update Property"

---

**Pro Tip**: Create properties as drafts first (uncheck "Publish"), perfect them, then publish when ready!
